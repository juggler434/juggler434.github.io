Ruby Basics

Variables A variable stores information. A variable must start with a lowercase letter or an underscore. You use '=' to set set a variable. Example: number = 12 Integers Integers are numbers without decimal points.
Floats Floats are numbers with decimal points Strings Strings are a collection of characters within quoatation marks. This can include numbers. ex: "This is a string with the number 12"
+ You can use + with numbers to add them together. You can also use + with strings to combine them. However, a string cannot be added to a number or vice versa. - - can be used with numbers to subtract them.
* * can be used with numbers to multiply them. It can also be used with a string and a number to repeat the string that many times. However, you cannot use this with a string and a string. / / can be used ot divide numbers. If integers are given, the result will be rounded down to the nearest integer.
== Returns true/false, compares if two things are equal to each other < and > return true/false, compare if two objects are greater than/less than each other
!= Returns true/false. Compares if two objects are not equal to each other <= and >= Returns true/false. Compare is two objects are equal to or greater/lesser than eachother
|| Stands for or, returns true if object on either side is true Ex: true || false is true && Stands for and, returns true if objects on boths sides are true Ex: true && false is false
Array An array is a collection of objects within brackets. Ex: my_array = [1, 'string', 8, 'words'] Hash A Hash is collection of key:value pairs. Ex: my_hash = {'one' => 1, 'twelve' => 12}
Array Position You can call on specific parts of an array by position. Position starts at 0. Ex my_array = [1, 4, 5] my_array[0] is 1 Hash values You can get or set a hash value by calling its key. Ex. my_hash = {'one' => 1, 'twelve' => 12} my_hash['one'] is 1
puts displays whats put after it as a string with a newline at the end print displays whats put after it litterally
