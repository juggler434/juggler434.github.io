Ruby Classes

Posted September 25, 2014

This week we've been learning about classes in Ruby. So what is a class? A class is like the framework in which objects are built. That sounds messy, so we're going to go through and build a class that will let us make some objects.

Since the new edition of Dungeons and Dragons just came out, I'm going to make a class that will let us build a dungeon room by room. We'll start by setting up our class.

          class DungeonRoom

          end

There we have our new class, but its not really doing much. We want to be able to set up our room with some basic statistics right when we make it, so for that we are going to create a method called initialize. This method will automatically be called when we create a new room.

          class DungeonRoom
            def initialize(length, width, height)
              @objects = []
              @monsters = []
              @length = length
              @width = width
              @height = height
            end
          end

Now when we make a new room, we will set a length, width and height, and it will create two empty arrays for objects and monsters. You might be wondering what the deal with those variables with @ in front of them is. Those are called instance variables, and they can be manipulated by any method within the class (as opposed to local variables which can't be messed with across methods). We're going to do more with those variables in a bit.

Now a dungeon room is pretty boring with out any monsters in it, so lets make a way to populate our room. We're going to create a new method for our class called add_monster

          class DungeonRoom
            def initialize(length, width, height)
              @objects = []
              @monsters = []
              @length = length
              @width = width
              @height = height
            end

            def add_monster(monster)
              @monsters << monster
            end
          end

Thats was easy, but our dungeon now needs some furniture, so lets add in a way to add some objects and a way to look at whats in our room.

          class DungeonRoom
            def initialize(length, width, height)
              @objects = []
              @monsters = []
              @length = length
              @width = width
              @height = height
            end

            def add_monster(monster)
              @monsters << monster
            end

            def add_object(object)
              @objects << object
            end

            def show
              puts "The room is #{@length} feet by #{@width} feet and #{@height} feet tall."
              puts "Monsters: #{@monsters}"
              puts "Objects: #{@objects}"
            end

          end

Allright, we now have our dungeon room creator class. Lets use it to make a room and check it out!

          entrance = DungeonRoom.new(10,10,20)
          entrance.add_monster("Orc")
          entrance.add_monster("Owlbear")
          entrance.add_object("Table")
          entrance.add_object("Pie")
          entrance.show

          The room is 10 feet by 10 feet and 20 feet tall.
          Monsters: ["Orc", "Owlbear"]
          Objects: ["Table", "Pie"]

There is the quick and dirty rundown of how to make a ruby class. We made a class named DungeonRoom, that we then used to make a room object. I hope this helps you understand a little more about clases in Ruby. Now go forth and code!
